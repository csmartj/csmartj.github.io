搭建个人博客

http://baixin.io/2015/08/HEXO%E6%90%AD%E5%BB%BA%E4%B8%AA%E4%BA%BA%E5%8D%9A%E5%AE%A2/
1 安装Node https://nodejs.org/en/ 下载新版 按步骤安装
2 安装git https://git-for-windows.github.io/ 下载新版 按步骤安装
3 安装 hexo 执行命令 npm install -g hexo 即会自动安装
4 创建博客，新建目录 blog1 进入：
hexo init 初始化 (hexo i blog1; cd blog1)
hexo g (hexo generate) 生成静态页面
hexo s (hexo server) 启动本地服务
浏览器输入http://localhost:4000 即可访问，如果长时间没有反应，那可能4000端口被占用了，可换其他端口 hexo s -p 5000 访问http://localhost:5000 即可
选择主题：git clone https://github.com/iissnan/hexo-theme-next themes/next
主题1 https://github.com/iissnan/hexo-theme-next
git checkout tags/v5.1.0

(http://blog.csdn.net/qq_31655965/article/details/53172143)
修改_config.yml theme: next
5 上传到github： npm install hexo-deployer-git –save
修改_config.yml (如果显示中文乱码，保存成 UTF-8 编码)
deploy:
repository: git@github.com:chenbourne/chenbourne.github.io.git
生成本地秘钥 ssh-keygen -t rsa 并copy到GitHub
测试 ssh -T git@github.com
生成 hexo d (hexo deploy)
https://chenbourne.github.io/ 可查看 (https://github.com/)
hexo clean && hexo g && hexo d
