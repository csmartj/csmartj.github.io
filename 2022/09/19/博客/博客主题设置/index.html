博客主题设置

Posted on 2022-09-19 Edited on 2018-12-13 In 博客

日志文件比较多的时候，博客了设置一些分类标签，能迅速的找到需要的东西，而且看起来比较舒服

下载主题

进入hexo创建博客的根目录，打开命令行，运行：
git clone https://github.com/theme-next/hexo-theme-next themes/next
这个命令就是下载next主题到当前目录themes下

启动主题

修改配置文件_config.yml
theme: next

主题设定

修改next/_config.yml,找到Scheme Settings，这里有四种外观
# Scheme Settings
#
-————————————————————–

# Schemes
#scheme: Muse 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
#scheme: Mist Muse 的紧凑版本，整洁有序的单栏外观
#scheme: Pisces 双栏 Scheme，小家碧玉似的清新
#scheme: Gemini 左侧网站信息及目录，块+片段结构布局`
想要哪个就把前面的#去掉就行

设置语言

修改配置文件_config.yml，将language设置成你所需要的语言，例如简体中文：
language: zh-Hans

设置菜单

修改next/_config.yml，找到menu
menu:
home: / || home
#about: /about/ || user
#tags: /tags/ || tags
#categories: /categories/ || th
archives: /archives/ || archive
#schedule: /schedule/ || calendar
#sitemap: /sitemap.xml || sitemap
#commonweal: /404/ || heartbeat
这里我们可以设置标签、分类。在这之前需要创建page
hexo new page 'tags'
hexo new page categories
在根目录下的source文件会生成tags、categories文件夹
分别修改tags和categories文件夹的index.md文件，新增type属性
-–
title: tags
date： 2018-08-08 12：12：12
type: tags
-–
categories的修改类似
在新建一篇博的时刻，增加tags和categories属性值，就能在tags和categories界面检索到文章了。

去掉文章后面的标签前面的#号

修改模板/themes/next/layout/_macro/post.swig，搜索 rel=”tag”>#，将#换成

动态背景

目前NexT主题最新的是V6.0版本，这个版本中可以有4种动态背景：
Canvas-nest
three_waves
canvas_lines
canvas_sphere
设置方法也很简单，直接设置里需要的动态背景为true。

侧边栏社交小图标设置

打开主题配置文件_config.yml，搜索social:, ||之后是在图标库中对应的图标。注意空格就行
social:
#GitHub: https://github.com/yourname || github
#E-Mail: mailto:yourname@gmail.com || envelope
#Google: https://plus.google.com/yourname || google
#Twitter: https://twitter.com/yourname || twitter
#FB Page: https://www.facebook.com/yourname || facebook
#VK Group: https://vk.com/yourname || vk
#StackOverflow: https://stackoverflow.com/yourname || stack-overflow
#YouTube: https://youtube.com/yourname || youtube
#Instagram: https://instagram.com/yourname || instagram
#Skype: skype:yourname?call|chat || skype
去掉前面的#就行
可以加上自己的
Google: http://www.google.cn/ || google
Github: https://github.com/ || github
baidu: https://www.baidu.com/ || baidu
Bing: https://cn.bing.com/ || Bing

博页面两侧的空白太多

编辑主题的source/css/_variables/custom.styl文件，新增变量：
// 修改成你期望的宽度
$content-desktop = 700px // 我的为800px，自己调整测试即可
// 当视窗超过 1600px 后的宽度
$content-desktop-large = 900px
我的电脑上的配置是
$content-desktop = 1000px
$content-desktop-large = 1200px

代码超出自动换行

修改 themes\next\source\css_common\scaffolding\base.styl
修改base.styl ,blockquote 样式下 添加 word-break: break-all;
